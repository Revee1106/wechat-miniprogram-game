# 炼丹系统 V1 进度说明

**日期：** 2026-03-31

本次按《问道长生》炼丹系统设计文档推进，采用“低成本首版”实现范围。

## 本次已完成

### 1. 炼丹运行态

- `RunState` 已支持 `alchemy_state`
- 已支持：
  - 炼丹术经验
  - 炼丹术等级
  - 炼丹术称号
  - 可见丹方列表
  - 当前炼制任务
  - 最近一次炼制结果
  - 成丹库存

### 2. 洞府联动

- `alchemy_room` 已从占位设施升级为真实功能入口
- 新增 `spirit_spring` 设施
- `灵田 + 灵泉 + 炼丹房` 已形成首版基础联动
- 灵泉月结算会产出 `spirit_spring_water`

### 3. 首版丹方

本次实装 6 个基础丹方：

- `辟谷丹`
- `养气丹`
- `养元丹`
- `宁神丹`
- `聚灵丹`
- `固元丹`

### 4. 炼制流程

- 已支持开炉
- 已支持“借灵泉开炉”辅材选项
- 开炉时会立即扣除材料
- 炼制按月份推进
- 每次推进时间时会同步推进炼制任务
- 到期后会结算：
  - 成丹 / 废丹
  - 品质
  - 熟练度增长

### 5. 品质与结果

- 已支持 3 档品质：
  - 下品
  - 中品
  - 上品
- 已支持废丹结果
- 已支持最近一次炼丹结果展示

### 6. 丹药使用

已支持成丹后直接服用，并实际作用于角色状态：

- `辟谷丹`：恢复少量寿元
- `养气丹`：直接增加修为
- `养元丹`：恢复气血
- `宁神丹`：降低状态惩罚
- `聚灵丹`：较高幅度增加修为
- `固元丹`：提高突破辅助值

### 7. API 与前端

- 新增 API：
  - `POST /api/run/alchemy/start`
  - `POST /api/run/alchemy/consume`
- 小程序 `run-store` 已支持：
  - `startAlchemy`
  - `consumeAlchemyItem`
- `pages/crafting` 已从占位页改为真实炼丹页
- 页面已支持：
  - 查看丹方
  - 开炉
  - 借灵泉开炉
  - 查看当前炉次
  - 查看最近结果
  - 查看成丹库存
  - 服丹

## 本次未完成

### 1. 丹炉系统

- 没有独立丹炉装备位
- 没有丹炉品质加成
- 没有丹炉耐久 / 修复

### 2. 高阶与复杂炼丹

- 没有筑基辅丹
- 没有残方 / 特殊事件池
- 没有批量炼制
- 没有炼丹队列
- 没有炸炉重度惩罚

### 3. 更深洞府联动

- 没有火脉 / 地火室
- 没有库房容量限制
- 没有按季节 / 地脉波动调整药材品质

### 4. 经济流转

- 没有丹药售卖
- 没有炼丹专门的交易接口
- 没有更细致的价值平衡后台
