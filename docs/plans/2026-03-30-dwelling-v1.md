# Dwelling V1 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a playable dwelling V1 loop with facility build/upgrade actions, monthly settlement, and a real dwelling page while keeping alchemy as a placeholder unlock.

**Architecture:** Extend `RunState` with dwelling facility runtime data and last-settlement logs, add dwelling build/upgrade APIs on the existing run workflow, and run dwelling settlement inside monthly time advance before event selection. Rework the mini-program dwelling page to consume the enriched run state and trigger build/upgrade actions through the shared store.

**Tech Stack:** FastAPI, Python dataclasses, in-memory repository, WeChat Mini Program JS/WXML/WXSS, Node-based frontend assertions, Pytest
