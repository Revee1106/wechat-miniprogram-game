# Economy V1 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 落地经济系统 V1 的最小闭环，包括资源定义目录、局内资源栈、单局清零、轮回点结算以及事件/突破对新经济口径的兼容。

**Architecture:** 新增 `app/economy` 领域层与 `config/economy` 本地配置层，核心循环继续保留在 `app/core_loop`，但通过 repository 和 service 读取统一经济模型。当前先用 JSON 配置承载资源目录和结算规则，同时避免服务层依赖具体存储格式，为未来迁移 MySQL 预留清晰边界。

**Tech Stack:** FastAPI, Python dataclasses, JSON config files, pytest
