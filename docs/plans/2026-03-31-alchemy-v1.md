# Alchemy V1 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a playable alchemy V1 loop with real recipes, time-based brewing, pill inventory, and immediate-use pill effects integrated with the dwelling system.

**Architecture:** Extend the run state with an `alchemy_state` runtime model, add an alchemy service that owns recipe availability, brewing jobs, result settlement, and pill consumption, and integrate brewing progress into monthly time advance. Upgrade the existing crafting page into the alchemy page and add a spirit spring dwelling facility as the basic auxiliary material source.

**Tech Stack:** FastAPI, Python dataclasses, in-memory repository, WeChat Mini Program JS/WXML/WXSS, Pytest, Node frontend assertions
